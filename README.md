## Commands

| KEY           | Action        |
| ------------- |:-------------:|
| `ESC`         | manual control     |
| `SPACE BAR`   | run         |
| `ENTER`         | jump |
| `CONTROL`         | crouch |
| `A`           | move left     |
| `D`           | move right    |
| `W`           | move forward  |
| `S`           | move backward |
| `‚Üí`           | turn right    |
| `‚Üê`           | turn left     |
| `shift`       | hide/display map|
| `MOUSE RIGHT CLICK`           | shoot |
| `MOUSE LEFT CLICK`           | aim - focus|
| `MOUSE SCROLL`      | change weapon |

you can customize the control keys inside the game settings
